# 1.2 艾森豪四象限：把「緊急」和「重要」分開
> 課程：時間管理實戰（範例課程）

## 本單元重點
- **緊急 ≠ 重要**。多數人整天在處理「緊急但不重要」，把「重要但不緊急」一直往後拖。
- 真正決定人生的，是第二象限（重要但不緊急）：健康、關係、學習、長期佈局。
- 目標是**擴大第二象限、壓縮第一象限、清掉第三與第四**。

## 內容筆記

**四個象限怎麼分**
| 象限 | 類型 | 怎麼處理 |
|---|---|---|
| 第一 | 緊急且重要（危機、期限） | 立刻做，但要檢討為何變緊急 |
| 第二 | 重要但不緊急（規劃、進修、運動） | 排進行事曆、優先保護 |
| 第三 | 緊急但不重要（多數會議、雜訊） | 授權、婉拒、批次處理 |
| 第四 | 不緊急也不重要（滑手機、瞎忙） | 盡量刪除 |

**關鍵洞察**
- 第一象限的事，很多其實是「以前的第二象限沒做」拖成的。**提前投資第二象限，就能減少未來的救火**。

## 可執行要點
- 把本週三件「重要但不緊急」的事，直接排進行事曆的固定時段。
- 下次有人塞事情給你，先問：這是第幾象限？

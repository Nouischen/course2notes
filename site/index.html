🚀 開始前的準備（一次就好）

開始使用：全程在 Claude Code 裡對話

你不用自己打任何指令、不用懂技術。用平常講話的方式跟 Claude Code 說，它就自己幫你安裝、開瀏覽器、下載、轉錄、做筆記。你唯一要動手的，是在它開的視窗登入你的課程平台（為了安全，只能你本人登）。過程中 Claude Code 會跳出權限確認（裝套件、開瀏覽器）——按「允許」即可，這是正常流程。

1

叫它安裝（一次就好）

🤖 對 Claude Code 說

在 Claude Code 裡打這句，它會自己下載安裝：

幫我把 github.com/Nouischen/course2notes 這個技能裝到 ~/.claude/skills/course2notes，之後我要用它把線上課程做成筆記

裝好後重開 Claude Code 一次讓技能生效，就一勞永逸。

2

叫它做筆記

🤖 對 Claude Code 說

把網址換成你的課程網址，打這句：

用 course2notes 把 https://你的課程網址 做成筆記

預設用你發問的語言整理。想指定就句尾加一句，例如：「筆記用英文」。

3

登入一次，剩下它全包

🖱️ 你只要登入

它會自動開一個瀏覽器視窗，你在裡面登入你的課程平台、打開那門課，跟它說「好了」。（你是在平台官方的登入頁輸入密碼，跟平常一樣；工具碰不到你的帳密。為什麼安全 ↓）接著它自動跑完：

偵察平台 → 下載音訊 → 轉逐字稿 → 整理筆記 → 產出漂亮 HTML，完成後雙擊 HTML 就是你的筆記。

每份筆記裡有什麼

隱私：我們只收數字

預設會回傳一筆「匿名使用計數」，幫我們判斷有多少人在用。只收：一組本機隨機碼（無法對應到你）、事件（安裝／完成筆記）、這次幾份筆記、平台類型、版本、時間。

絕對不收：你的筆記內容、課程名稱、網址、任何個資或檔案。程式碼開源可自查。關閉：設 COURSE2NOTES_TELEMETRY=off。

🎙️ 關於轉錄：有 NVIDIA 顯卡或 Apple Silicon Mac 時，音檔在本機轉錄、不離開你的電腦；其他情況（Intel Mac／AMD／無顯卡）用 OpenAI API 模式時，你的音檔會上傳給 OpenAI 轉錄（適用 OpenAI 的 API 資料政策），但作者端一樣收不到任何東西。

運作原理（想深入了解再看）

它不是一套「固定爬蟲」，而是讓你的 Claude Code 臨場看懂每個平台、自己想辦法抓——所以能適應很多站。整條流程：

偵察平台→抓單元清單→下載音訊→whisper 轉逐字稿→AI 整理筆記→產出漂亮 HTML
支援哪些平台？

影片掛在常見服務、且沒有硬 DRM 加密的平台多半可行；純文字課也能整理。

類型支援
Vimeo / SoundCloud / YouTube / HLS 串流的平台
（如知識衛星、Teachify 類、LearnDash 類…）		✅ 可
純文字／文章型課程✅ 可
硬 DRM 加密串流（部分大型平台）✘ 不可

⚖️ 本工具僅供你為「自己已合法購買、且有權存取」的課程製作個人學習筆記之用。散布的是工具，不是任何課程內容。請遵守你所在平台的服務條款與著作權法，勿散布或轉售下載／產生的內容。使用行為與後果由使用者自行負責。

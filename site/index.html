怎麼運作

你只要動一次滑鼠登入，剩下的它自己來。

1

登入你的課程平台

用一個「除錯模式」的瀏覽器登入你買課的平台（工具會給你指令）。工具不碰你的密碼。

2

貼上課程網址

在你的 Claude Code 說「把這門課做成筆記」。它會自己偵察平台、抓出所有單元。

3

自動下載＋轉逐字稿＋做筆記

抓音訊 → whisper 轉逐字稿 → AI 整理成結構化重點筆記（修簡繁、抓框架、附可執行要點）。

4

得到一份漂亮 HTML

一個檔案、任何瀏覽器打開就看、可搜尋、可列印成 PDF。不需要 Notion、不需要註冊。

支援哪些平台？

它不是固定爬蟲，而是讓 AI 臨場看懂每個平台——所以能適應很多站，但不保證每一個。

類型支援說明
影片掛 Vimeo / SoundCloud / YouTube / HLS 的平台✅ 可如 LearnDash、Teachify 類、知識衛星等常見架構
純文字 / 文章型課程✅ 可直接萃取內文重點
硬 DRM 加密串流✘ 不可部分大型平台的加密影片抓不到

🚀 設定精靈

填幾個選項，直接產生你要貼的指令，不用背。

隱私：我們只收數字

預設會回傳一筆「匿名使用計數」，幫我們判斷有多少人在用。只收：

  • 一組本機隨機產生的匿名安裝碼（無法對應到你）
  • 事件（安裝／完成一份筆記）、這次幾份筆記、平台類型、版本、時間

絕對不收：你的筆記內容、課程名稱、課程網址、任何個人資料或檔案。程式碼開源，你可以自己檢查。關閉：設 COURSE2NOTES_TELEMETRY=off 或 config 裡 "telemetry": false。

⚖️ 本工具僅供你為「自己已合法購買、且有權存取」的課程製作個人學習筆記之用。散布的是工具，不是任何課程內容。請遵守你所在平台的服務條款與著作權法，勿散布或轉售下載/產生的內容。使用行為與後果由使用者自行負責。
